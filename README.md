# WebassemblyCompiler

## Description

Compiles J-- code to WAT.

## WAT

WAT stands for WebAssembly Text Format. For more information, see https://developer.mozilla.org/en-US/docs/WebAssembly

## J--

A subset of Java. For more information, see J--LanguageSpec.txt.

## Requirements

Ubuntu 18.04

Bison 3.0.4

Flex 2.6.1
